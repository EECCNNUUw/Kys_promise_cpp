# KYS Promise Reborn (金庸群侠传前传 C++ 重构版)

本项目旨在将基于 Pascal/Delphi 开发的《金庸群侠传前传》底层引擎迁移至 C++。通过重新实现核心逻辑与渲染层，提升程序的运行效率与后续开发的扩展性。

## 项目现状

目前已完成基础框架的搭建与部分核心功能的 C++ 移植工作。重构过程严格遵循原版逻辑，以确保二进制数据与游戏行为的一致性。

### 已实现功能

*   **底层渲染层**：基于 SDL3 重新实现了图形渲染接口，支持硬件加速。
*   **资源解析系统**：实现了对原版 `.grp`、`.idx`、`.pic` 等二进制资源文件的直接读取与解析。
*   **UI 框架重构**：
    *   完成了 ESC 环形主菜单的数学建模与逻辑实现。
    *   实现了包括物品、武学、状态、系统、技能、队友在内的六大次级菜单界面。
    *   适配了繁体编码（Big5/UTF-8）的文本渲染。
*   **全局状态管理**：引入了统一的单例管理模式，对游戏全局变量与内存状态进行规范化管理。

### 开发中/待完善

*   **战斗逻辑**：正在移植原版的战斗数值计算与 AI 行动逻辑。
*   **指令/事件系统**：待完善对原版复杂脚本指令集的解析与执行。
*   **音频系统**：MIDI 与音效的跨平台播放适配。

## 技术选型

*   **语言标准**：C++20
*   **多媒体库**：SDL3, SDL3_image, SDL3_ttf
*   **构建系统**：CMake
*   **运行平台**：Windows (目前主要开发平台)

## 开发说明

本项目通过 CMake 进行工程管理。在配置好 SDL3 及其扩展库环境后，可直接编译生成执行文件。目前主要用于底层重构的技术验证，游戏逻辑仍在逐步完善中。
